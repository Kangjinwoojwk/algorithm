* 문제 3

  n = 1일때, 부분 집합은 해당 원소가 있는 것과 없는 것, 2가지이다.

  n + 1일때, 부분집합은 n 일때의 부분집합에 추가된 원소가 있는 것과 없는 것 두가지가 되므로

  원소갯수 n인 집합의 부분집합수 * 2 = 원소갯수 n + 1인 집합의 부분집합수 이다.

  n이 1일때 2개 이므로, 부분집합의 종류는 2^n개이다.
